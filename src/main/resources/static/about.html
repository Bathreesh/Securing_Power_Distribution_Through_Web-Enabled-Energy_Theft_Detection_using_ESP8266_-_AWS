About GridSecure

GridSecure is an IoT-based Smart Energy Theft Detection System designed to combat electricity theft in power distribution networks using real-time monitoring and cloud-based analytics.

🤝 Industry Internship Collaboration

This project was developed as part of an internship program conducted by Hexhive Solutions from 21 January 2026 to 03 February 2026.

The team worked collaboratively to design, develop, and deploy a web-enabled energy theft detection system using ESP8266 microcontrollers and AWS cloud services. The project focuses on reducing power losses, improving real-time monitoring, and enhancing grid security through IoT integration.

📍 Hexhive Solutions, Pondicherry 📧 hexhivesolutions@gmail.com

🎯 Our Mission

To provide utility companies with an affordable, scalable solution for detecting and preventing energy theft, reducing losses and ensuring fair billing for all consumers.

🔧 Technology Stack

Hardware: ESP8266, PZEM-004T Energy Meter, Current Transformers
Backend: Spring Boot, JWT Authentication, H2 Database
Cloud: AWS IoT Core, Lambda, DynamoDB

📊 Key Features

  • Real-time voltage & current monitoring
  • Anomaly detection algorithms
  • Instant theft alerts via SMS/Email
  • Historical data analytics
  • Multi-meter dashboard
