Architecture Layers

📟

Hardware Layer

IoT Sensor Node

ESP8266 NodeMCU microcontroller connected to PZEM-004T energy meter for real-time voltage, current, and power monitoring. Data is transmitted via WiFi to AWS cloud.

ESP8266 PZEM-004T Arduino IDE MQTT
☁️

Cloud Layer

AWS Infrastructure

AWS IoT Core receives MQTT messages from sensors. Lambda functions process data and run theft detection algorithms. DynamoDB stores readings for historical analysis.

AWS IoT Core Lambda DynamoDB SNS
🖥️

Backend Layer

REST API Server

Spring Boot application provides RESTful APIs for the frontend. JWT authentication secures endpoints. H2 database stores user data and alert configurations.

Data Flow

1

Sensor Reading

PZEM-004T measures voltage, current, power, and frequency. ESP8266 reads values via serial interface every 5 seconds.

2

MQTT Transmission

ESP8266 publishes JSON payload to AWS IoT Core via secure MQTT connection with TLS encryption.

3

Anomaly Detection

Lambda function analyzes readings against baseline. Detects theft patterns like low voltage + high current (bypass) or meter tampering.

4

Alert Generation

If anomaly detected, SNS sends instant notifications via SMS/email. Alert is logged in database for review.

5

Dashboard Update

Web dashboard fetches data via REST API. Real-time charts and alert panels display current system state.
